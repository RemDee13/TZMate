# Screenshots

Capture these screenshots for the public prerelease.

## Required Screenshots

### 1. `01-contacts.png`

- Menu bar popover open
- Contacts tab selected
- At least 3 contacts:
  - Germany / Berlin
  - United Kingdom / London
  - United States / Los Angeles
- One favorite contact
- One Good time status
- One Late night or Evening status if possible

### 2. `02-add-contact-phone-lookup.png`

- Add contact form open
- Phone lookup result for `+1` or `+61`
- Shows multiple countries or multiple time zones

### 3. `03-converter.png`

- Time Converter tab selected
- Example: Bangkok 15:00 = London 09:00
- Result visible
- Copy button visible

### 4. `04-settings.png`

- Settings tab selected
- Theme picker visible
- Time format picker visible
- Default time zone visible

### 5. `05-widget.png`

- macOS widget on desktop or in Notification Center
- Shows favorite contacts

## Style

- Use light mode for main screenshots unless dark mode looks better.
- Keep the menu bar popover centered and clean.
- Avoid showing private contacts, emails, or real phone numbers.
- Use sample names only.
- Crop images cleanly.
- Prefer PNG.
