# TZ Mate v0.1.0 Alpha

## Overview

TZ Mate v0.1.0 Alpha is an early developer preview of a native macOS menu bar app and widget for checking client local time, converting time zones, and planning international calls.

This alpha is source-only. No signed or notarized `.dmg` is included yet.

## What Is Included

- Native macOS menu bar app
- AppKit status item and SwiftUI popover
- Contacts with local time, time difference, and status labels
- Favorite contacts for widget display
- Phone code and country name lookup
- Local country, city, phone code, and IANA time zone data
- Time converter
- Settings for theme, time format, Launch at Login, and updates
- Quit action from Settings and the status item menu
- WidgetKit widget extension
- App Group shared storage abstraction
- Sparkle-based update-checking foundation
- Local-first storage

## Installation

Build from source using Xcode.

No public binary download is included with this alpha. A signed and notarized `.dmg` is planned for a later GitHub Release.

## Build From Source

1. Clone the repository.
2. Open `TZMate.xcodeproj` in Xcode.
3. Select your Development Team for both targets:
   - `TZMate`
   - `TZMateWidgetExtension`
4. Enable App Groups for both targets:

```text
group.com.remdee.tzmate
```

5. Build and run the `TZMate` scheme.

Unsigned local verification builds can also be run with:

```sh
xcodebuild -scheme TZMate -configuration Debug build CODE_SIGNING_ALLOWED=NO
xcodebuild -scheme TZMateWidgetExtension -configuration Debug build CODE_SIGNING_ALLOWED=NO
```

## Known Limitations

- This is an early developer preview.
- No signed or notarized `.dmg` is included yet.
- Unsigned local builds may trigger macOS Gatekeeper warnings.
- Launch at Login may require a properly signed local build.
- The app icon is temporary.
- Screenshots are placeholders.
- Public binary updates are not active until an appcast and signed archives are published.

## Widget Note

The widget may require a locally signed build with the same Development Team and App Group enabled for both targets. If the widget does not appear, clean the build folder, rebuild, run the main app, then search for `TZ Mate` in the macOS widget gallery.

## Updates Note

Sparkle support is included as a foundation for future GitHub Releases. Production updates require:

- Developer ID signing
- Apple notarization
- Sparkle-signed release archives
- A published HTTPS `appcast.xml`
- A confirmed final appcast URL

The Mac App Store is not planned for the initial version.

## Privacy

- No account required
- No backend
- No analytics
- No tracking
- Contacts and settings are stored locally on the user's Mac
- The widget reads local App Group storage only
- Optional update checks use Sparkle and the configured HTTPS appcast URL
