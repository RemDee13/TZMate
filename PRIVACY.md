# Privacy

TZ Mate is local-first.

- No account is required.
- No backend is used.
- No analytics are collected.
- No tracking is included.
- Contacts and settings are stored locally on the user's Mac.
- The widget reads local shared storage through the configured App Group.
- Optional update checks use Sparkle and the configured HTTPS appcast URL.
- Update checks are for app releases only; they are not analytics or tracking.

If cloud sync is added in the future, it should be optional and clearly documented.
